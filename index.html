Notas

Como usar:
1. Arraste arquivos diretamente para a grade quadriculada;

2. Use o botão Sincronizar para mapear de uma vez sua pasta local "Dexi" e guardar tudo de forma permanente;

3. Clique nas categorias para filtrar. Clique em "Ver Todos" para resetar o filtro.

Codex

Solte arquivos das suas notas e fontes diversas aqui como cards flutuantes. Você pode arrastá-los livremente pela grade.
